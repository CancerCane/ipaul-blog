Every year I tweak my dev setup. Here's where things stand in 2026.

## The essentials

- **Editor**: VS Code is still the workhorse. I've tried others, but the extension ecosystem keeps pulling me back.
- **Terminal**: I've been using Warp lately. The AI integration is genuinely useful for someone who doesn't memorize every CLI flag.
- **Version Control**: Git + GitHub. Nothing fancy.
- **Languages**: Ruby (Jekyll sites), Python (scripts and automation), JavaScript (when I have to).

## What changed this year

The biggest shift was leaning into AI-assisted development. Not as a replacement for understanding code, but as a way to move faster on the parts I already understand conceptually.

I used to spend 30 minutes looking up syntax I'd forgotten. Now I describe what I want and verify the output. The time savings add up.

## My advice for fellow late-starters

Don't try to learn everything. Pick a stack and go deep. You can always branch out later. The fundamentals — version control, debugging, reading docs — those transfer everywhere.

Start building something real on day one. Tutorials are fine, but nothing teaches like a project that matters to you.
